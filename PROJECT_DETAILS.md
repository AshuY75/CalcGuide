# CalcGuide - Project Documentation
**"Sahi Hisab, Sahi Faisla" (Right Calculation, Right Decision)**

## 1. Project Overview
CalcGuide is a high-performance, SEO-optimized Progressive Web App (PWA) built for the Indian market. It provides a suite of financial, construction, and utility calculators designed to rank high on Google and offer immediate value to users without signups or paywalls.

*   **Live URL**: `https://calcguide.com` (Production-ready config)
*   **Primary Audience**: Indian investors, home buyers, students, and rural users (Village Interest).
*   **Core Philosophy**: Speed, Accuracy, and Trust.

---

## 2. Technology Stack
*   **Framework**: React 18 + Vite (Fastest build tool)
*   **Styling**: Tailwind CSS (Mobile-first, utility classes)
*   **Routing**: React Router DOM v6 (Centralized, Lazy Loading)
*   **SEO**: `react-helmet-async` (Head management), `schema-dts` (Structured Data)
*   **Build**: ESBuild (via Vite)
*   **Deployment**: Static Site Generation (SSG) ready (Vercel/Netlify compatible)

---

## 3. Architecture & Core Systems

### A. Routing System (Deep Linking)
We use a centralized routing architecture to support deep, SEO-friendly URLs.
*   **Config File**: `src/routes/paths.js` (Single source of truth)
*   **Router**: `src/routes/AppRoutes.jsx` (Lazy loaded components)
*   **URL Pattern**: `/calculators/{category}/{calculator-slug}`
    *   *Example*: `/calculators/investment/sip-calculator`

### B. SEO Engine
Every page is strictly optimized for Google crawling:
1.  **Meta Tags**: Dynamic Title, Description, Canonical URL via `<Helmet>`.
2.  **Schema Markup**: JSON-LD generated by `SchemaGenerator` component (FAQPage, Calculator, HowTo).
3.  **Sitemap**: Auto-generated via `node generate-sitemap.js` containing all deep links.
4.  **Content Structure**: Semantic HTML (H1 -> H2 -> H3) with specialized content sections.

### C. UI/UX Design System
*   **Header**: Sticky, with "Typewriter" animation and categorized Mega Menu.
*   **Footer**: SEO-rich links categorized by utility.
*   **Layout**: `MainLayout` wrapper with automated scroll-to-top.
*   **Calculator Cards**: Interactive entry points with "Popular", "New", "Trending" tags.
*   **Mobile Experience**: Fully responsive touch targets and drawers.

## 6. Development Workflow

### Standard Commands
*   `npm run dev`: Start local server (Port 5173).
*   `npm run build`: Compile for production (`/dist` folder).
*   `npm run preview`: Test production build locally.
*   `node generate-sitemap.js`: Update `sitemap.xml` after adding routes.

### How to Add a New Calculator
1.  **Create Page**: Add `<NewCalc>.jsx` in `src/pages/`.
2.  **Add Route Path**: Define URL constant in `src/routes/paths.js`.
3.  **Register Route**: Add path & lazy import to `src/routes/AppRoutes.jsx`.
4.  **Update Sitemap**: Add URL to `generate-sitemap.js` array.
5.  **Link UI**: Add card in `Home.jsx` or Menu in `Header.jsx`.
