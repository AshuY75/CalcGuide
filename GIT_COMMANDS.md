# Git Commands Reference

Here are the commands to save your changes and push them to GitHub.

### 1. Check Status
See which files have changed.
```powershell
git status
```

### 2. Add Changes
Stage all modified files for the commit.
```powershell
git add .
```

### 3. Commit Changes
Save the changes with a descriptive message.
```powershell
git commit -m "Your updated message here"
```

### 4. Push to Remote
Upload your commits to GitHub.
```powershell
git push
```

---

### Emergency Commands (Use with Caution)

**Discard all local changes (Revert to last commit):**
```powershell
git reset --hard
```

**Pull latest changes from GitHub:**
```powershell
git pull
```
